Welcome to SQLAlchemy-HANA
==========================

SQLAlchemy-HANA makes it possible to use the SAP HANA database with
SQLAlchemy. Thereby it is getting easier to create Python applications like
web applications with frameworks like Flask and the features of SAP HANA.

.. warning::

    Please notice that sqlalchemy-hana is not an official SAP product and
    is not covered by SAP support.

Contribute

If you found bugs or have other issues, you are welcome to create a GitHub Issue.

`Stack Overflow <http://stackoverflow.com/>`_ question with tag
`sqlalchemy <http://stackoverflow.com/questions/tagged/sqlalchemy>`_ and
